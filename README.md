# Sanikinetics
*Sobre o projeto

O Sanikinekts é um projeto conceitual que propõe o uso de placas cinéticas instaladas em rodovias para gerar energia elétrica a partir da passagem dos veículos.

Como exemplo de aplicação, utilizamos a Rodovia Raposo Tavares para estimar o potencial de geração de energia. A energia produzida poderia ser utilizada para abastecer uma estação de tratamento de água e saneamento, contribuindo para soluções mais sustentáveis para a cidade. Caso haja excedente de energia, ele poderia ser direcionado para o abastecimento urbano.

*Objetivo do trabalho

O objetivo do trabalho proposto pelo professor era desenvolver um site informativo apresentando a ideia do projeto e explicando seu funcionamento.

Para isso, foi desenvolvido um site utilizando HTML, com o intuito de organizar as informações de forma clara e acessível.
